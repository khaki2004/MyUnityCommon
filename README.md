Yggdrasil
  
  NFT技術を使い
    ・二次販売
    ・先払い、後でいつでも配送
  を便利にするためのアプリ
  
  バーチャルの街を歩いてショッピングもできる（3Dモード、2Dモード）
  NFT（商品交換チケット）を先に購入→いつでもNFTと商品を交換できるため保管の手間がなく、「やっぱりいらない」となったらNFTを誰かに売れるので初期購入ハードルが下がる
  
  全ての商品には取引数があり、新品は１。取引数が２以上のものはオリジナル以外の誰かが転売したものである。（中古）
    ・取引数１しか商品と交換できない
    ・任意の取引数の商品を任意の数集めると景品をプレゼント
  などの指定も出品者ができる（オリジナルのみ）
  
  商品の種類は
    ・NFTチケット（リアル商品と交換するためのチケット）
    ・リアル商品（NFTチケットを渡して実物を買う、配送するもの）
    ・バーチャル商品（電子書籍、アート、3Dデータなど）
  の3つ
